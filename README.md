# EnDi
- It is an open source search engine written in **python** with using the **Django** framework.
___
## Technology
- The search is carried out by the name of the site.
- Sites are added to the database manually using the **html** page.
___
## Used
1. Python and Django
2. HTML, CSS
3. MySQL
